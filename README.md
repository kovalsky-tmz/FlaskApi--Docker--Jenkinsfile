# FlaskApi-Jenkins-Docker
 Docker image : kowal20x7/flaskapi

Api:
POST   /login    - to get jwt token
GET    /posts
POST   /posts    - {'title','content'}
PUT    /post/{id}
DELETE /post/{id}
GET    /post/{id}
GET    /users
POST   /users    - to add new user {'username','password'}
PUT    /user/{id}
DELETE /user/{id}
GET    /user/{id}
